## Day 1 — รีบูทธุรกิจใน 2 นาที

```
คุณเป็นที่ปรึกษาธุรกิจมืออาชีพ ฉันต้องการ "รีบูท" ธุรกิจของฉันอย่างรวดเร็ว

ธุรกิจของฉัน: [ชื่อธุรกิจ / ประเภทธุรกิจ]
ปัญหาหลักตอนนี้: [ระบุปัญหา 1-2 อย่าง]
งบประมาณที่มี: [ระบุงบ]

ช่วยสรุปให้ฉัน 3 สิ่งที่ต้องทำทันทีเพื่อกลับมามีกำไรได้เร็วที่สุด พร้อมระบุว่าแต่ละอย่างใช้เวลากี่วัน และต้องลงทุนเท่าไร
```

## Day 2 — วิเคราะห์ตลาดด้วย AI ทันที

```
ช่วยวิเคราะห์ตลาดสำหรับธุรกิจของฉันแบบเจาะลึก:

ธุรกิจ: [ชื่อ/ประเภท]
สินค้า/บริการ: [รายละเอียดสั้น ๆ]
กลุ่มลูกค้าเป้าหมาย: [อายุ / อาชีพ / ความสนใจ]
ภูมิภาค: [ออนไลน์ / จังหวัด / ทั่วประเทศ]

วิเคราะห์ให้ครบ:
1. ขนาดตลาด (TAM / SAM / SOM)
2. คู่แข่งหลัก 3-5 ราย พร้อมจุดแข็ง-จุดอ่อน
3. เทรนด์ตลาด 12 เดือนข้างหน้า
4. ช่องว่างในตลาดที่ฉันสามารถเข้าไปได้
5. คำแนะนำ positioning ที่เหมาะสม
```

## Day 3 — เขียนอีเมลหาลูกค้าเก่า

```
ช่วยเขียนอีเมล Re-engagement สำหรับลูกค้าเก่าที่หายไป:

บริษัทของฉัน: [ชื่อบริษัท]
สินค้า/บริการ: [อธิบายสั้น ๆ]
ลูกค้าเก่าที่จะส่งถึง: [ประเภทลูกค้า / ซื้อสินค้าอะไรครั้งล่าสุด]
ข้อเสนอพิเศษที่มี: [ส่วนลด / ของแถม / บริการใหม่]

เขียนอีเมล 3 เวอร์ชัน:
1. แบบอบอุ่น-ส่วนตัว (Warm & Personal)
2. แบบตรงประเด็น-มีข้อเสนอ (Offer-focused)
3. แบบถามไถ่-ไม่ขายของ (Check-in)

แต่ละเวอร์ชัน: Subject line + เนื้อหาอีเมล (ไม่เกิน 150 คำ) + CTA
```

## Day 5 — สร้างแผนการตลาด 16 หน้า

```
สร้างแผนการตลาดครบถ้วนสำหรับธุรกิจของฉัน:

ธุรกิจ: [ชื่อ / ประเภท]
สินค้า/บริการหลัก: [อธิบาย]
งบการตลาดต่อเดือน: [จำนวนบาท]
เป้าหมาย 3 เดือน: [ยอดขาย / จำนวนลูกค้าใหม่ / อื่น ๆ]
ช่องทางที่มีอยู่แล้ว: [เช่น Facebook / Line / Website / หน้าร้าน]

สร้างแผนครอบคลุม 16 หัวข้อ:
1. Executive Summary
2. การวิเคราะห์ SWOT
3. กลุ่มเป้าหมาย (Buyer Persona)
4. Positioning & Messaging
5. กลยุทธ์ช่องทางการตลาด
6. Content Strategy (รายเดือน)
7. Social Media Plan
8. Email Marketing Plan
9. Paid Advertising Plan
10. SEO/SEM Strategy
11. Influencer/Partnership Plan
12. Promotion Calendar
13. KPI และตัวชี้วัด
14. งบประมาณแบ่งตามช่องทาง
15. Timeline การดำเนินงาน
16. แผนรับมือวิกฤต
```

## Day 6 — วางแผนการเงินแบบเฉลียวฉลาด

```
ช่วยสร้างแผนการเงินธุรกิจแบบ Smart Financial Planning:

ธุรกิจ: [ชื่อ / ประเภท]
รายได้ปัจจุบัน (ต่อเดือน): [จำนวนบาท]
ค่าใช้จ่ายหลัก: [ระบุหมวดหลัก ๆ + ยอดโดยประมาณ]
กำไรสุทธิ (ต่อเดือน): [จำนวนบาท]
เป้าหมายการเงิน 1 ปี: [เช่น เพิ่มกำไร 30% / สร้าง Emergency Fund / ลดหนี้]

วิเคราะห์และวางแผนให้:
1. สุขภาพการเงินปัจจุบัน (ดี/เสี่ยง/วิกฤต + เหตุผล)
2. จุดคุ้มทุน (Break-even Point)
3. Cash Flow Projection 6 เดือน
4. โอกาสลดต้นทุนโดยไม่กระทบคุณภาพ
5. แนวทางเพิ่มรายได้ 3 วิธี
6. แผนสำรองฉุกเฉิน (Emergency Fund)
7. เป้าหมายรายเดือนที่วัดผลได้
```

## Day 7 — คัดเลือกผู้จำหน่ายด้วย Claude

```
ช่วยสร้างระบบคัดเลือกผู้จำหน่าย (Vendor Selection) ให้ธุรกิจของฉัน:

สินค้า/วัตถุดิบที่ต้องการ: [ระบุ]
ปริมาณที่ต้องการ (ต่อเดือน): [จำนวน]
งบประมาณต่อหน่วย/ต่อเดือน: [จำนวนบาท]
เงื่อนไขสำคัญ: [เช่น ส่งภายใน X วัน / คุณภาพ ISO / มีหลักประกัน]

สร้างให้ครบ:
1. เกณฑ์คัดเลือก (Scorecard) พร้อมน้ำหนักคะแนนแต่ละด้าน
2. รายการคำถามสัมภาษณ์ผู้จำหน่าย (10 ข้อ)
3. เอกสารที่ต้องขอจากผู้จำหน่าย
4. ตาราง Comparison สำหรับเปรียบเทียบ 3-5 ราย
5. สัญญาณเตือน (Red Flags) ที่ควรระวัง
6. Template อีเมลขอใบเสนอราคา (RFQ)
```

## Day 8 — อัปเกรดโมเดลธุรกิจให้สร้างรายได้

```
ช่วยวิเคราะห์และอัปเกรด Business Model ของฉัน:

ธุรกิจปัจจุบัน: [อธิบายสั้น ๆ]
รูปแบบรายได้ปัจจุบัน: [ขายครั้งเดียว / Subscription / Commission / อื่น ๆ]
จุดแข็งที่มี: [ทักษะ / ฐานลูกค้า / ทรัพยากร]
เป้าหมาย: [เพิ่มรายได้ / สร้าง Passive Income / Scale ขึ้น]

วิเคราะห์และแนะนำ:
1. จุดอ่อนใน Business Model ปัจจุบัน
2. โมเดลรายได้เพิ่มเติมที่เหมาะกับธุรกิจฉัน (3-5 ตัวเลือก)
3. กลยุทธ์ Upsell / Cross-sell / Bundling
4. วิธีสร้าง Recurring Revenue
5. แผน 90 วันในการเพิ่มช่องทางรายได้ใหม่
6. ตัวอย่างธุรกิจที่คล้ายกันที่ทำได้สำเร็จ
```

## Day 9 — สรุปประชุมอัตโนมัติ

```

[วางข้อความ/transcript การประชุมที่นี่]

---หรือถ้ายังไม่มี transcript ใช้ prompt นี้---

สร้าง Template สรุปการประชุมมาตรฐานสำหรับทีมของฉัน:

ระยะเวลาประชุม: [นาที]

สร้าง:
1. Template สรุปการประชุมมาตรฐาน (พร้อม Sections ที่ชัดเจน)
2. Prompt ที่ทีมใช้วาง transcript แล้วสรุปได้ทันที
3. ระบบติดตาม Action Items

## Day 10 — ออกแบบโครงสร้าง Pitch Deck

```
ช่วยออกแบบ Pitch Deck ที่น่าสนใจสำหรับธุรกิจของฉัน:

ชื่อธุรกิจ: [ชื่อ]
สินค้า/บริการ: [อธิบายสั้น ๆ]
กลุ่มเป้าหมาย: [นักลงทุน / พาร์ทเนอร์ / ลูกค้าองค์กร]
จุดเด่นหลัก: [USP ของธุรกิจ]
ต้องการอะไร: [เงินลงทุน / ความร่วมมือ / สัญญา]

ออกแบบ Pitch Deck 10-12 สไลด์:
1. Title Slide — ชื่อ + Tagline ที่ดึงดูด
2. Problem — ปัญหาที่แก้ (ข้อมูล + เรื่องราว)
3. Solution — วิธีแก้ปัญหา (ชัดเจน ไม่เกิน 3 bullet)
4. Product/Service Demo — แสดงผลิตภัณฑ์
5. Market Opportunity — ขนาดตลาด
6. Business Model — รายได้มาจากไหน
7. Traction — ผลลัพธ์ที่มีอยู่แล้ว
8. Team — ทีมและความเชี่ยวชาญ
9. Competition — แผนภาพเปรียบเทียบ
10. Financials — ประมาณการรายได้ 3 ปี
11. The Ask — ต้องการอะไร ใช้ทำอะไร
12. Contact & Next Steps

พร้อมเนื้อหาสำหรับแต่ละสไลด์
```

## Day 11 — เพิ่มยอดขายผ่านช่องทางออนไลน์

```

ธุรกิจ: [ชื่อ / ประเภท]
ช่องทางปัจจุบัน: [Facebook / Line / Shopee / Lazada / Website / อื่น ๆ]
ยอดขายออนไลน์ต่อเดือน: [จำนวนบาท]
งบโฆษณาต่อเดือน: [จำนวนบาท]
สินค้า/บริการที่ขาย: [รายการหลัก]

วางแผนให้:
1. วิเคราะห์ช่องทางที่ให้ ROI ดีที่สุดสำหรับธุรกิจฉัน
2. กลยุทธ์เพิ่มยอดขาย 30% ใน 60 วัน
3. Content Calendar รายสัปดาห์ (ทำได้คนเดียว)
4. แผนโฆษณา (Paid Ads) อย่างประหยัด
5. เทคนิค Conversion Rate Optimization
6. ระบบ Follow-up ลูกค้าที่ยังไม่ตัดสินใจซื้อ
```

```

```

## Day 16 — สร้างแผนแม่บทโซเชียลมีเดีย

```
สร้าง Social Media Master Plan สำหรับธุรกิจของฉัน:

ธุรกิจ: [ชื่อ / ประเภท]
แพลตฟอร์มที่ใช้: [Facebook / Instagram / TikTok / LinkedIn / YouTube / X]
จำนวน Followers ปัจจุบัน: [แต่ละแพลตฟอร์ม]
ทีมคอนเทนต์: [คนเดียว / มีทีม X คน]
งบโฆษณาต่อเดือน: [จำนวนบาท]
เป้าหมาย 3 เดือน: [Followers / Engagement / ยอดขาย]

สร้างแผนครบ:
1. Brand Voice & Tone Guidelines
2. Content Pillars (4-5 หัวข้อหลัก)
