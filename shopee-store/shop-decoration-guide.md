## 3. หลักการลงสินค้า (Add SKU) — ตาม PDF หน้า 36-40

### รูปแบบชื่อสินค้า
```
[แบรนด์] [ชื่อสินค้า] [รุ่น/รายละเอียด] [จุดเด่น]
```
**ตัวอย่าง:**
```
TCiAP TOEIC Hybrid Learning คอร์สออนไลน์+ออนไซต์ 36 ชั่วโมง [CEFR 2025]
```
- ❌ **ห้ามใช้อีโมจิ** ในชื่อสินค้า
- ✅ คำโฆษณาให้ใส่ใน **วงเล็บสี่เหลี่ยม []**
- ✅ ไม่เกิน **120 ตัวอักษร**

### รูปภาพสินค้า
- **ขนาด:** 500×500 pixel ขึ้นไป (1:1 ratio)
- **จำนวนต่อสินค้า:** อย่างน้อย 3 รูป (สูงสุด 9 รูป + วิดีโอ)
- **รูปหน้าปก:** พื้นหลัง **สีขาว** เท่านั้น
- ถ่ายหลายมุมมอง: หน้าปกคอร์ส, เนื้อหา, ผู้สอน, ตารางเรียน, Testimonial

### คำอธิบายสินค้า (Description)
ควรประกอบด้วย:
1. ชื่อและรายละเอียดคอร์ส (ชั่วโมง, รูปแบบ, เวลา)
2. จุดเด่น/ข้อดี
3. เหมาะสำหรับใคร
4. วิธีรับสินค้า/คอร์ส (ติดต่อ LINE @toeic)
5. ข้อควรระวัง/หมายเหตุ
- ❌ **ห้ามใส่ลิงค์** ไปยังเว็บไซต์อื่น

## 5. การตั้งค่าการจัดส่ง (สำหรับสินค้าดิจิทัล)

เนื่องจากสินค้าเป็นคอร์สออนไลน์ (Digital):

- ใส่หมายเหตุใน Description ว่า "ส่งข้อมูลผ่าน LINE @toeic หลังยืนยันออเดอร์"

## 6. เทคนิคเพิ่มยอดขาย — ตาม PDF หน้า 56-68

### สูตรยอดขาย
```
ยอดขาย = จำนวนคนดูสินค้า × อัตราการซื้อ
```

**เพิ่มจำนวนคนดู:**
- เข้าร่วมแคมเปญ Shopee อย่างสม่ำเสมอ
- ใช้ฟังก์ชัน **Boost** ใน Seller Centre ทุกวัน
- ใช้ **Shopee Ads** (My Ads) เพื่อซื้ออันดับการค้นหา
- แชร์ร้านค้าไปยัง Social Media

**เพิ่มอัตราการซื้อ:**
- ตอบแชทลูกค้าภายใน **15 นาที** (คะแนนตอบกลับ ≥ 75%)
- รูปภาพสวยงาม มีคุณภาพ
- คำอธิบายสินค้าละเอียดและน่าเชื่อถือ
- มีส่วนลดและโค้ดส่วนลดดึงดูดใจ

### Preferred Seller Requirements (ร้านค้าแนะนำ) — ตาม PDF หน้า 63
| เกณฑ์ | ค่าที่ต้องการ |
|------|------------|
| ยอดสั่งซื้อ/เดือน | ≥ 100 ออเดอร์ |
| ผู้ซื้อไม่ซ้ำ/เดือน | ≥ 15 คน |
| คะแนนร้านค้า | ≥ 4.5 ดาว |
| อัตราส่งสินค้าล่าช้า | ≤ 5% |
| อัตราคำสั่งซื้อไม่สำเร็จ | ≤ 5% |
| คะแนนการตอบกลับ | ≥ 75% |

## 7. การตั้งค่า Shop Decoration (หน้าร้าน)

เข้าที่ **Seller Centre → ตั้งค่าร้านค้า → ตั้งค่าร้านค้า**

### ลำดับ Section ที่แนะนำ:
1. **Banner Carousel** — รูป Promotion หลัก 5 ใบ (600×300px)
2. **Product Highlight** — สินค้าขายดี/แนะนำ 4 ชิ้น
3. **Category Display** — แยกตามหมวดคอร์ส (TOEIC / มหาวิทยาลัย / องค์กร)
4. **Products by Category** — แสดง 8 สินค้าต่อหมวด
5. **YouTube Video** — วิดีโอแนะนำสถาบัน TCiAP
